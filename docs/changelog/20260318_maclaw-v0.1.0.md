# 20260318 — MacLaw v0.1.0 (Initial Release)

## Summary

macOS-native AI agent runtime. Swift 建構、零 Node.js 依賴、Keychain 密鑰管理、launchd daemon、Telegram bot 介面。支援 Codex 和 Claude 雙 backend 切換。

## Added

### Core
- Swift CLI 入口點 (`maclaw`) with ArgumentParser
- `maclaw setup` — 互動式初始設定精靈
- `maclaw gateway run|stop|restart|status` — gateway 生命週期管理
- `maclaw daemon install|uninstall|status` — launchd plist 管理
- `maclaw secrets set|delete|list` — Keychain 密鑰管理（透過 `security` CLI，零彈窗）

### Backend System
- Backend protocol 抽象層（可擴充）
- `maclaw backend status|set|install|login` — backend 管理
- **Codex backend** — `codex exec --full-auto`，session resume via `thread_id`
- **Claude backend** — `claude -p --output-format json`，session resume via `--resume`
- Per-backend session 儲存（`sessions-codex.json` / `sessions-claude.json`）
- 切換 backend 時保留各自的 session history

### Telegram Bot
- URLSession + NWPathMonitor 原生網路層（解決 OpenClaw 的 IPv6/sleep 問題）
- 長輪詢 (long-polling) with 自動重連
- 訊息分割（>4096 字元自動切割）
- 持續 typing indicator（每 4 秒更新直到回覆）
- Access control: `dmPolicy`, `groupPolicy`, `allowFrom`, `groupAllowFrom`
- AI 驅動的群組回覆判斷（`--json-schema` structured output → `shouldRespond`）
- 群組訊息含 sender name（`Che Cheng: 訊息`）

### Telegram Commands
- `/start` — 歡迎訊息
- `/help` — 指令列表
- `/ping` — 健康檢查
- `/status` — gateway 狀態（PID、uptime、backend、model）
- `/model` — 顯示完整 backend config（model、reasoning effort 等）
- `/model set <name>` / `/model reset` — runtime model 切換
- `/permissions` — 顯示目前權限模式
- `/permissions safe|full` — 切換安全/完全存取模式
- `/reset` — 清除 session，開始新對話
- `/whoami` — Telegram user info

### Configuration
- `~/.maclaw/maclaw.json` — 主設定檔
- `@keychain:` prefix — Keychain 密鑰引用
- `allowedTools` — 可設定的工具白名單（scoped patterns）
- `~/.maclaw/workspace/CLAUDE.md` — bot identity + 行為規則
- Backend config 自動從 `~/.codex/config.toml` 或 `~/.claude/settings.json` 讀取

### Cron Scheduler
- Recurring jobs (`every 1h`) + one-shot jobs (`at 2026-03-17T10:00:00Z`)
- 指數退避（30s → 1m → 5m → 15m → 60m）
- State 持久化到 `~/.maclaw/cron-state.json`
- 失敗通知 via Telegram

### Testing
- 13 unit tests: PKCE、message chunking、cron parser、config loader

## Changed

- 從直接 API call 重構為 CLI delegation（設計原則：delegate to CLI, don't reimplement）
- 移除 OAuth PKCE 實作（改用 backend CLI 自帶的 auth）
- 移除 LLMProvider/ModelRouter（改用 `codex exec` / `claude -p`）

## Tech Stack / Files

- Swift 6.0, macOS 14+, Apple Silicon
- swift-argument-parser 1.3+（唯一外部依賴）
- URLSession + NWPathMonitor（原生網路）
